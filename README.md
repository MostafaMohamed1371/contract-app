## نظام إدارة العقود والقضايا

تطبيق Laravel لإدارة **العقود، القضايا، والإنذارات/الإشعارات القانونية** بواجهة عربية حديثة ولوحة تحكم (Dashboard)، مع رفع ملفات (PDF / مستندات) وتنبيه آلي قبل انتهاء العقود.

---

## المزايا الرئيسية

- **إدارة العقود (`contracts`)**
  - رقم العقد، اسم الجهة، نوع العقد (صيانة / تقنية / توريد / استشارة …).
  - تاريخ التوقيع، مدة العقد (نص حر)، تاريخ الانتهاء.
  - قيمة العقد، حالة العقد (فعال – منتهي – ملغى).
  - رفع نسخة PDF موقعة وتخزينها في `storage/app/public/contracts`.
  - حقل `notify_before_days` لتحديد عدد الأيام قبل الانتهاء لإطلاق التنبيه (افتراضيًا 30 يومًا).

- **إدارة القضايا (`legal_cases`)**
  - رقم الدعوى، رقم القضية الداخلي.
  - المحكمة، نوع الدعوى، المدعي / المدعى عليه.
  - المحامي المسؤول، تاريخ الجلسة القادمة، حالة القضية (مستمرة – مؤجلة – مكتسبة الدرجة القطعية).

- **إدارة الإنذارات والإشعارات (`legal_notices`)**
  - أنواع الإنذار/الإشعار: إنذار تسديد، إنذار فسخ عقد، إنذار تقصير، مطالبة مالية، أخرى.
  - ربط اختياري بعقد أو بقضية.
  - رفع مرفق واحد (PDF / DOC / XLS …) وتخزينه في `storage/app/public/notices`.

- **لوحة تحكم حديثة**
  - Sidebar مع أيقونات وروابط: الرئيسية، العقود، القضايا، الإنذارات.
  - كروت إحصائية (إجمالي العقود / القضايا / الإنذارات).
  - آخر العقود، الجلسات القادمة للقضايا، أحدث الإنذارات.

- **تنبيهات انتهاء العقود**
  - أمر مجدول `contracts:notify-expiring` يفحص العقود يوميًا.
  - يعتمد على `end_date` و `notify_before_days` لكل عقد.
  - حاليًا يسجل العقود التي أوشكت على الانتهاء في الـ log، ويمكن تطويره لاحقًا لإرسال بريد/إشعار.

- **اللغة والتحقق من المدخلات**
  - لغة الواجهة الافتراضية: العربية (`APP_LOCALE=ar`).
  - ملف رسائل تحقق عربي مخصص في `lang/ar/validation.php`.
  - فحص نوع وحجم الملفات، وصلاحية التواريخ، وإجبار الحقول الأساسية (مثل رقم العقد، نوع الإشعار، المستلم…).

## المتطلبات

- PHP 8.2 أو أحدث.
- MySQL (أو MariaDB) مع قاعدة بيانات باسم `contract`.
- Composer.

---

## خطوات التشغيل محليًا

1. **تنزيل الحزم**

   ```bash
   composer install
   ```

2. **إعداد ملف البيئة**

   انسخ ملف البيئة الافتراضي وعدّل إعدادات قاعدة البيانات حسب بيئتك:

   ```bash
   cp .env.example .env
   ```

   تأكد من القيم الأساسية في `.env`:

   ```env
   DB_CONNECTION=mysql
   DB_HOST=127.0.0.1
   DB_PORT=3306
   DB_DATABASE=contract
   DB_USERNAME=root
   DB_PASSWORD=

   APP_LOCALE=ar
   APP_FALLBACK_LOCALE=en
   SESSION_DRIVER=database
   ```

3. **توليد مفتاح التطبيق**

   ```bash
   php artisan key:generate
   ```

4. **تشغيل المايجريشن وإنشاء الجداول**

   ```bash
   php artisan migrate
   ```

5. **ربط مجلد التخزين بالـ public (لخدمة الملفات المرفوعة)**

   ```bash
   php artisan storage:link
   ```

6. **تشغيل الخادم التطويري**

   ```bash
   php artisan serve
   ```

   ثم افتح المتصفح على:

   - لوحة التحكم: `http://localhost:8000/`
   - العقود: `http://localhost:8000/contracts`
   - القضايا: `http://localhost:8000/legal-cases`
   - الإنذارات: `http://localhost:8000/legal-notices`

## منطق تنبيه انتهاء العقود

- حقل `notify_before_days` في جدول `contracts` يحدد عدد الأيام قبل تاريخ نهاية العقد `end_date` التي يُعتبر عندها العقد “قرب انتهاء”.
- أمر Artisan مخصص في `App\Console\Commands\NotifyExpiringContracts`:

  - يجمع العقود ذات `end_date` غير null.
  - يحسب تاريخ اليوم + `notify_before_days`.
  - إذا كان `end_date` يطابق هذا التاريخ، يتم تسجيل العقد في السجل (`storage/logs/laravel.log`).

- تم جدولة الأمر في `bootstrap/app.php`:

  ```php
  ->withSchedule(function (Schedule $schedule): void {
      $schedule->command('contracts:notify-expiring')->daily();
  })
  ```

- على السيرفر الإنتاجي يجب تشغيل الـ scheduler عبر Cron (مثال):

  ```bash
  * * * * * cd /path/to/contract-app && php artisan schedule:run >> /dev/null 2>&1
  ```

## ملاحظات حول الواجهة

- الواجهة مبنية باستخدام **Tailwind CSS CDN** مع خط عربي حديث (`Tajawal`).
- قالب رئيسي واحد في `resources/views/layouts/app.blade.php`.
- جميع الشاشات (`contracts`, `legal-cases`, `legal-notices`, `dashboard`) ترث من هذا القالب وتستفيد من نفس الـ sidebar والتصميم.

## تطوير مستقبلي مقترح

- إضافة صلاحيات مستخدمين (أدوار: مدير نظام، محامي، موظف عقود).
- إرسال بريد إلكتروني أو إشعارات مدمجة بدل الاكتفاء بالـ log لتنبيهات انتهاء العقود.
- إضافة فلاتر وبحث متقدم للعقود والقضايا والإنذارات.
